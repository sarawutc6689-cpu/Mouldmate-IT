ระบบจัดการข้อมูลพนักงาน

คำแนะนำการตั้งค่า Google Apps Script:

  1. เปิด Google Apps Script Editor
  2. ไปที่ Deploy > New deployment
  3. เลือก Type: Web App
  4. ตั้งค่า: Execute as: Me, Who has access: Anyone
  5. กด Deploy และคัดลอก URL ที่ได้
  6. เปลี่ยนค่า scriptUrl ในโค้ด JavaScript เป็น URL ของคุณ

ค้นหาข้อมูลพนักงาน

กรอกรหัสพนักงาน (Username) เพื่อค้นหาและแก้ไขสถานะการทำงาน

กำลังค้นหาข้อมูลพนักงาน...

รายละเอียดข้อมูลพนักงาน
รหัสพนักงาน (คอลัมน์ A)
ชื่อ-นามสกุล (คอลัมน์ B)
สถานะ (คอลัมน์ C)
การดำเนินการ (คอลัมน์ D)
วันที่เริ่มงาน (คอลัมน์ E)
รหัสผ่าน (คอลัมน์ F)

คำอธิบายการใช้งาน:

  • กรอกรหัสพนักงานในช่องค้นหาเพื่อดึงข้อมูลจาก Google Sheets
  • คอลัมน์ A, B, C, E, F เป็นข้อมูลอ่านอย่างเดียว ไม่สามารถแก้ไขได้
  • คอลัมน์ D (การดำเนินการ) สามารถแก้ไขได้โดยเลือก TRUE หรือ FALSE
  • หลังจากเลือกค่าแล้วกดปุ่ม "บันทึกการเปลี่ยนแปลง" เพื่ออัปเดตข้อมูล
  • ข้อมูลจะถูกบันทึกลงใน Google Sheets ผ่าน Google Apps Script
